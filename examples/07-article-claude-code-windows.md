Claude Code ставится за 10 минут. Большинство об этом не знает.

Все слышали про Claude, но думают, что «Claude Code» - это что-то для программистов, для линукса, для людей с бородой и тремя мониторами.

Нет. Это просто инструмент в терминале, который ставится через одну команду. И на Windows тоже работает - вроде норм, я сам пользуюсь.

В этой статье - как поставить всё с нуля. Без опыта. Под Windows. Прямо в VSCode.

✨ Поехали ✨

Глава 1. Что вам нужно перед стартом. /SETUP/

Два инструмента, без которых ничего не запустится:

1. Node.js - это движок, который запускает javascript на твоём компе, плюс идёт с npm (менеджером пакетов). npm - это магазин, из которого ставятся инструменты типа Claude Code.
2. VSCode - редактор кода, в нём встроенный терминал, туда и будем вводить команды.

Больше ничего особенного не нужно.

Глава 2. Ставим Node.js. /NODE/

Заходишь на nodejs.org и качаешь версию LTS (это стабильная, не трогай Сurrent).

Открываешь установщик, жмёшь «Далее» везде - там ничего трогать не надо. По умолчанию Node.js сам пропишется в PATH, то есть будет доступен из любой папки в терминале.

После установки открой терминал - Win+R, пишешь cmd, Enter - и проверь:

node --version
npm --version

Должны появиться цифры, типа v22.0.0 и 10.x.x. Если появились - готово. Если ошибка - перезапусти комп и попробуй снова.

Глава 3. Открываем терминал в VSCode. /TERMINAL/

Если VSCode ещё не стоит - качаешь с code.visualstudio.com, ставишь, открываешь.

Дальше открываешь папку с проектом: File → Open Folder. Или просто любую папку - это не принципиально для установки.

Терминал открывается через Ctrl+` (это тильда, кнопка слева от единицы). Или через меню Terminal → New Terminal.

В нижней части экрана появится строка ввода. Это и есть терминал, туда вводим команды дальше.

Глава 4. Ставим Claude Code. /INSTALL/

Одна команда:

npm install -g @anthropic-ai/claude-code

Флаг -g значит «globally» - ставим не в папку проекта, а на весь компьютер. После этого команда claude будет доступна из любой папки.

Установка занимает 30-60 секунд. Ждёшь, не трогаешь.

Проверяешь:

claude --version

Должна вылезти версия. Всё, стоит.

Глава 5. Первый запуск. /START/

Теперь вводишь просто:

claude

Первый раз оно попросит залогиниться через браузер - это нормально, авторизация через Anthropic аккаунт. Если аккаунта нет - регистрируешься на claude.ai, это бесплатно на старте.

После авторизации ты попадаешь в интерактивный чат прямо в терминале.

Несколько примеров что можно сделать сразу:

Есть файл main.py и хочешь понять что там: «Объясни мне что делает этот код» - и дальше вставляешь код.

Есть баг и не понимаешь почему: «Вот ошибка: [текст ошибки]. Что не так и как исправить?»

Хочешь написать что-то с нуля: «Напиши скрипт который читает файл csv и выводит первые 10 строк»

Работает как обычный Claude, только он видит твои файлы и может их редактировать прямо из терминала.

Глава 6. Частые проблемы. /FIXES/

npm не найден после установки Node.js - перезапусти VSCode полностью, иногда PATH не подхватывается сразу.

Ошибка EACCES или permissions при npm install -g - открой VSCode от имени администратора (правая кнопка на иконке → «Запуск от имени администратора»).

claude не найден после установки - проверь что npm install -g прошёл без ошибок, повтори. Если всё равно нет - попробуй npx @anthropic-ai/claude-code вместо claude.

Windows Defender заблокировал что-то - разреши, это не вирус, это npm-пакет.

Итог

Claude Code - это обычный чат, только в терминале и с доступом к файлам.

Ничего сложного. Node.js + одна команда npm, и всё работает.
