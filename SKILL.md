# Emil's Voice

Портативный движок голоса: тон, ритм, лексика и структурные привычки Эмиля, выжатые из корпуса его публичных постов и стримов. Язык голоса — **только русский**.

## Применение (рантайм)

Когда нужно написать или переписать текст голосом Эмиля:

1. Прочитай `voice-profile.md` — дистиллированный стиль-гайд (регистры, живые маркеры, универсальные правила, чек-лист).
2. Если задача диалоговая (ответ в переписке, постановка задачи, фидбэк) — дополнительно прочитай `chat-style.md`.
3. Прочитай 2–3 файла из `examples/` — эталонные реальные посты, держи их перед глазами как образец ритма и тона.
4. Пиши/переписывай, применяя профиль. **API при этом не вызывается** — всё уже в файлах.
5. Перед выдачей прогони по чек-листу самопроверки в конце `voice-profile.md`.

## Структура

```
SKILL.md            ← этот файл (рантайм)
voice-profile.md    ← публичный голос: регистры, маркеры, правила (грузится всегда)
chat-style.md       ← чат-стиль: задачи, фидбэк, переписка (грузится по необходимости)
examples/           ← эталонные реальные посты
config.json         ← хэндлы и источники
tools/collect.mjs   ← сборщик корпуса через ScrapeCreators API
```
