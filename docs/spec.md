# Спека: emils-voice

**Дата:** 2026-06-01

## Цель
Портативный Claude Code скилл, который заставляет Claude писать/переписывать текст голосом Эмиля Латыпова в любом проекте.

## Подход
Дистиллированный профиль + эталоны (вариант A). Корпус собирается один раз через ScrapeCreators API → анализ выжимает `voice-profile.md` + `examples/`. В рантайме грузятся только эти файлы, без вызовов API.

## Источники
- Threads: `latypov.emil` — `/v1/threads/user/posts`
- LinkedIn: `rrotatew` (`https://www.linkedin.com/in/rrotatew`) — `/v1/linkedin/profile`
- YouTube: 2 стрима — `/v1/youtube/video/transcript`
- База API: `https://api.scrapecreators.com`, заголовок `x-api-key`
- Язык голоса: только русский

## Компоненты
- `config.json` — хэндлы и лимиты
- `tools/collect.mjs` — сборщик (Node, без зависимостей, `--probe` для проверки)
- `voice-profile.md` — дистиллированный стиль-гайд (сердце)
- `examples/` — 5–10 эталонных реальных постов
- `corpus/` — сырьё (gitignore)
- `SKILL.md` — инвокабельный скилл

## Качество
Holdout-тест: отложить 2–3 реальных поста, сгенерировать на их тему, сравнить стилистическое попадание.

## Деплой
Отдельный репозиторий в GitHub Эмиля.

## Открытые зависимости
- `SCRAPECREATORS_API_KEY` (для сборки корпуса)
- авторизация `gh` (для создания и пуша репо)
- точные имена query-параметров эндпоинтов подтверждаются первым живым запросом (`--probe`); collect.mjs написан терпимо к разным именам полей ответа
